# Sorteador de Números

Projeto desenvolvido para gerar números aleatórios dentro de um intervalo definido pelo usuário, com opção de não repetir números e diversas validações para garantir uma ótima experiência.

## 📁 Estrutura do Projeto

- `index.html` — Estrutura principal da página.
- `style.css` — Estilos e responsividade.
- `scripts.js` — Funcionalidades e lógica do sorteio.
- `assets/` — Imagens e ícones utilizados.

## 🚦 Regras e Requisitos

### Estrutura da Página

- Título principal e seção explicando o funcionamento do sorteador.
- Área para o usuário inserir:
  - Quantidade de números a serem sorteados.
  - Valor mínimo do intervalo.
  - Valor máximo do intervalo.
  - Opção de não repetir números.

### Funcionalidades

- Sorteio de números aleatórios conforme os parâmetros definidos.
- Opção para não repetir números sorteados.
- Botão "Sortear" para iniciar o sorteio.
- Resultados exibidos de forma clara e animada.
- Resultados atualizados a cada novo sorteio.

### Validações

- Não permite campos vazios.
- O valor máximo deve ser maior que o valor mínimo.
- A quantidade de números não pode ser maior que o intervalo disponível (quando a opção de não repetir estiver marcada).
- Mensagens de erro amigáveis são exibidas quando necessário.

### Exibição dos Resultados

- Números sorteados aparecem na tela de forma clara e organizada.
- Animação ao exibir os resultados.
- Resultados são atualizados a cada sorteio.
